# Foodbiz one-pager — The Brief Box

This folder is a self-contained Claude Code session. Open it in Claude Code, point Claude at `CLAUDE.md`, and it has everything it needs to start building.

## The concept in one line
A single-page website that asks "*What are you planning?*" and reshapes itself around the visitor's answer.

## What Claude Code will do
1. Read `CLAUDE.md` (instructions specific to this build).
2. Read `PRD.md` (full product spec).
3. Read `brand-voice.md` (voice + banned vocabulary).
4. Read `content.md` (placeholder copy to use verbatim).
5. Initialise a Next.js 15 + Tailwind + Framer Motion project in this folder.
6. Build the site, deploy to a Vercel preview URL, and drop a `STATUS.md` when done.

## What's in here
- `CLAUDE.md` — instructions for Claude Code (start here)
- `PRD.md` — full product spec
- `brand-voice.md` — voice + banned vocab
- `business-context.md` — Foodbiz background
- `content.md` — placeholder copy
- `logos/` — brand marks (foodbiz-parent, caterpillar, gummy-gourmet, foodbiz-weddings-legacy as reference)
- `README.md` — this file

## What you'll get
A Vercel preview URL showing the Brief Box one-pager. Designed to be shareable with Amar and Kush for approval. Moira and Pillar Live use typographic placeholders (real logos pending).
